Instructions on setting up Main and Footer navigation for your website.

Top Level Navigation

  1. If you have not created the page you want to add to the navigation, do that now.
  2. From the Add Page to Navigation dropdown, select your page. The page will appear at the bottom of the navigation area seen above.
  3. Using the arrows move the menu item up/down to position where you want it to appear.
  4. Click on the Save Navigation button.
  5. Return to your website and refresh page. Your new navigation/page should now appear and be ready live.

Secondary Level Navigation

  1. If the secondary menu item you are creating is being associated with a new Top Level menu item, you will need to create the top level [Placeholder] menu item first. From the Add Page to Navigation dropdown select the [Placeholder] option. It will appear in the navigation area seen above. You are required to give this [Placeholder] a name. Enter the navigation name.
  2. Create the page you want to add to the navigation.
  3. From the Add Page to Navigation dropdown, select your page. The page will appear at the bottom of the navigation area seen above.
  4. Using the arrows move the new menu item up/down to position below the [Placeholder] item you created in the fist step. Press the right arrow to position your menu item as a child of the top level [Placeholder] item.
  5. Click on the Save Navigation button.
  6. Return to your website and refresh page. Your new child navigation/page should now appear and be ready live.
